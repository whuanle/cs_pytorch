## 导读

本教程通过使用 C# ，学习数学基础、Pytorch 框架、深度学习等，目前还在编写中。

- 教程中每个小节都有代码示例
- 深入原理，讲解深层知识
- 由易到难，从入门到掌握
- 循序渐进，一步步学习，一步步拓展知识面
- 内容完整、齐全，可以系统式学习
- 大量代码示例和场景实践
