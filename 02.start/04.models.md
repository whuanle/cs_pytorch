# 使用预训练模型



添加引用：

```xml
	<ItemGroup>
		<PackageReference Include="TorchSharp" Version="0.103.1" />
		<PackageReference Include="TorchSharp-cuda-windows" Version="0.103.1" />
		<PackageReference Include="TorchVision" Version="0.103.1" />
	</ItemGroup>
```
